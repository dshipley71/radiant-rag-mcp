Full feature reference across ingestion, retrieval, generation, and infrastructure

Feature What it supports MCP tool / env key
INGESTION
Documents PDF, DOCX, MD, HTML, code (.py .js .ts .go .rs …), JSON/JSONL, images (VLM) — hierarchical parent/child or flat storage ingest_documents
Web & GitHub Single-page or depth-crawled URLs; GitHub repos (all markdown); basic-auth private sites; automatic link following ingest_url
Video MP4, MKV, MOV, WebM, AVI + any yt-dlp URL (YouTube, Twitter/X, Vimeo, TikTok, 1 000+ sites) — Whisper for audio, VLM frame-window analysis for silent video ingest_video
Audio MP3, WAV, M4A, FLAC, OGG, AAC, OPUS, WMA, AIFF — Whisper transcription; rejects non-audio paths before processing ingest_audio
RETRIEVAL
Hybrid search Dense vector + BM25 sparse, fused with Reciprocal Rank Fusion; best general-purpose mode; hierarchical auto-merge promotes sibling chunks to full parent context search_knowledge mode=hybrid
Dense search HNSW cosine similarity via sentence-transformers/all-MiniLM-L12-v2 (384-dim, local, no API key); swappable to all-mpnet-base-v2 (768-dim) search_knowledge mode=dense
BM25 search Exact-term keyword matching; best for identifiers, code symbols, technical strings; in-memory index rebuilt from vector store on demand search_knowledge mode=bm25
Cross-encoder rerank ms-marco-MiniLM-L12-v2 rescores top-k fused candidates; CPU ~3 s or GPU T4 ~0.3 s; configurable top-k funnelling RADIANT_RERANKING_BACKEND_DEVICE
GENERATION & QA
Full RAG pipeline Query rewrite → expand → retrieve → rerank → context eval → synthesize → critic (confidence 0–1) → citation tracking; any OpenAI-compatible LLM backend query_knowledge
Simple query Skips planning, decomposition, critic, and citation agents; lower latency (~5–8 s on Ollama cloud); ideal for factual lookups simple_query
Multi-turn conversation Session IDs persist query history across turns; pipeline receives prior context automatically; in-memory or backend-persisted start_conversation + query_knowledge
Video / audio summary VideoSummarizationAgent: map-reduce chapter summaries + overall + key topics; three verbosity presets (brief / standard / detailed); works on transcripts and VLM frame captions ingest_video / ingest_audio summarize=true
INFRASTRUCTURE
LLM backends Ollama (local or cloud), OpenAI, vLLM, HuggingFace local; configurable timeout, retries, model name; env-var override at runtime RADIANT_LLM_BACKEND_*
VLM (vision) Qwen2-VL-2B local or any Ollama-cloud vision model; used for image captions and silent-video frame-window analysis; fails gracefully when unavailable RADIANT_VLM_*
Storage backends ChromaDB embedded (dev / Colab, no service); Redis Stack with HNSW (production, low latency, binary quantization 3.5× memory reduction) RADIANT_STORAGE_BACKEND
Quantization Binary or Int8 on both backends; 1M docs (384-dim): 1 536 MB → 432 MB; retrieval 50–100 ms → 5–10 ms; 95–96% accuracy retained quantization.precision
Monitoring Prometheus metrics exporter + OpenTelemetry tracing; per-agent latency, status, and custom metrics; exposable at /metrics endpoint PrometheusMetricsExporter
